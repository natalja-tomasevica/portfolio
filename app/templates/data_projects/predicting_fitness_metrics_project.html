Fitness Records Analysis and Prediction with Machine Learning¶

In this project, I will analyse the data about fitness metrics and build a machine learning model.

Attributes:

  • Duration: Length of the workout session (in minutes)
  • Pulse: Average heart rate during the session (in beats per minute)
  • Maxpulse: Maximum heart rate reached during the session (in beats per minute)
  • Calories: Estimated calories burned during the session

3. Creating machine learning models¶

3.1 Building a linear regression model¶

Comments¶

I chose to create a linear regression model for predicting calories burnt because it is a simple, interpretable model that works well when both the target variable (calories burnt) and the features (duration, max pulse, and average pulse) are numeric. The model assumes a linear relationship between these variables, allowing me to understand how each feature contributes to the predicted calories. Linear regression serves as an excellent baseline model to evaluate the relationship between workout characteristics and calorie burnt.

I will split the dataset into:

  • The training set (80%), which is used to fit the model
  • And the testing set (20%), which is used to evaluate the model’s performance

Comments¶

Interpretation of coefficients

  • 5.81: This coefficient corresponds to the duration. It indicates that for each additional minute of duration, the model predicts an increase of approximately 5.81 calories burnt, holding the other variables constant
  • 0.50: This coefficient corresponds to the average pulse. It suggests that for each additional bpm in the average pulse, the model predicts an increase of about 0.50 calories burnt, holding the other variables constant
  • 2.83: This coefficient corresponds to the max pulse. It indicates that for each additional bpm in the max pulse, the model predicts an increase of approximately 2.83 calories burnt, keeping the other variables constant

Interpretation of intercept

  • The intercept of -427.5 indicates the predicted calories burnt when all features (duration, average pulse, and max pulse) are at zero. In this context, the negative intercept does not have a meaningful interpretation, as having zero duration and pulses isn't realistic in a fitness context

Summary

I can determine that the linear regression model performed well, because since the RMSE is relatively small compared to the average calories burnt, indicating that the model has a reasonable predictive capability.

The above scatter plot shows how closely the predicted values align with the actual values of calories burnt. The dashed red line represents the ideal scenario where the predicted values perfectly match the actual values. This line is at a 45-degree angle, meaning that if a point lies on this line, the predicted and actual values are the same. Many of the blue points are close to the dashed red line, which indicates that the model performs reasonably well in predicting calories burnt for most data points, suggesting that the model can generalize well to the data.

There are a few points above the red line, indicating that for these observations, the predicted calories burnt are lower than the actual values. Specifically, some points at the higher end (around 800-1000 calories) show significant discrepancies between actual and predicted values. These outliers may indicate cases where the model struggles to make accurate predictions, possibly due to unique characteristics in those data points not captured in the model.

In general, since the points are generally scattered around the red line, I can conclude that the linear regression model has good predictive capabilities.

# Calculate residuals
residuals = y_test - y_pred_lr

plt.scatter(y_pred_lr, residuals, color = 'blue', alpha = 0.6)
plt.axhline(y = 0, color = 'red', linestyle = '--')
plt.title('Linear regression: residual plot')
plt.xlabel('Predicted calories burnt')
plt.ylabel('Residuals')
plt.grid()

# Histogram of residuals
plt.subplot(1, 2, 2)
plt.hist(residuals, bins = 20, color = 'blue', alpha = 0.6)
plt.axvline(x = 0, color = 'red', linestyle = '--')
plt.title('Linear regression: histogram of residuals')
plt.xlabel('Residuals')
plt.ylabel('Frequency')
plt.grid()

# Show the plots
plt.tight_layout()
plt.show()

Comments¶

Residuals are the differences between the actual values and the predicted values generated by a regression model.

Residual plot analysis (left side):

  • Distribution around zero: The residuals seem to be scattered around the horizontal dashed red line (which represents zero). This indicates that there is no systematic error in the predictions, suggesting that the linear regression model is appropriate for this dataset.
  • Randomness: The residuals do not exhibit any clear patterns or trends, which is a good sign. For example, if there was a curve or systematic trend, it might have indicated that a linear model is not suitable, and a non-linear model could be better.
  • Spread of residuals: While the majority of the residuals are close to zero, there are a few outliers (such as the points below -100 and above 100). These could be points where the model significantly over- or under-predicted the actual calories burnt.
  • Outliers: There are some larger residuals for certain predicted values, especially around 200-400 calories. This suggests that the model's predictions for these values may need further investigation.

Histogram of residuals (right side):

  • Distribution shape: The histogram shows a distribution that is somewhat skewed to the left, with a concentration of residuals near zero. A normally distributed set of residuals is ideal for linear regression. However, this histogram indicates some departures from normality, particularly with the presence of residuals in the range of -100 to 0.
  • Frequency: The highest frequency appears around -50 to -100, which suggests that the model is slightly under-predicting calories burnt for a number of observations. This could also be indicative of some bias in the model.
  • Spread: The histogram shows that while most residuals are close to zero, there are significant tails, indicating that while the model performs well on average, there are cases where it is off by a large margin.

Summary

The residual plot and histogram together suggest that the linear regression model is fairly appropriate for this data, but there may be outliers or certain regions where the model underperforms.
